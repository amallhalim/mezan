# Quickstart: Calculator

## Running Locally

1. Start the development server:
   ```bash
   npm run dev
   ```
2. Navigate to `http://localhost:3000/calculator`

## Testing the Phases

- **Phase 1**: Verify that adding an item to the plate, editing its quantity, and clearing the plate all work without errors.
- **Phase 2**: Inspect the page using browser dev tools. Verify that background styles are applied via Tailwind classes (e.g. `bg-gradient-to-r`) and no inline style attributes exist for colors.
- **Phase 3**: Verify that a skeleton loader appears when the page first loads, simulating data fetch delay.
